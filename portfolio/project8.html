Windows Applications

Summary

Various MS Windows application design

Roles

UI/UX Designer, Python programmer

This is a collection of additional designs I created for various Windows-based products.

Cash Letter Utility

The following design was for an application that lets users collate a large collection of individual branch deposits so they can be ingested into the bank's core system.

Focus Groups Utility

This design was for an application that allowed admins to organize branch deposits according to 'focus groups', which were based on geographic location and time.

Python Test Utility

The next design was a utility I wrote in Python with the wxPython UI library to simulate errors coming from the check processor. The screens themselves could be tailored by device, model, and error code to provide the operator with specific images/videos and recovery/maintenance information.

Designer Configuration Tool for NCompass

This last design was for an application that provided a front-end for many different configuration files. This gave the administrator both a single place from which to configure the application as well as provide structure, inline help, and validation.
